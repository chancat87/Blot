Converting the Jekyll theme

Checklist for converting a Jekyll theme (e.g. Minima) to Blot after you’ve done the initial setup in Convert a Jekyll template to Blot.

1. Getting the source

Download the theme from GitHub and unzip (or clone) as in the main guide.

2. Convert the _config.yml file

Use an online converter to map Jekyll config to Blot’s package.json.

  • Map paginate integer to page_size.

3. Move files in _includes to root

Move files in _includes to the template root directory and add an underscore at the start of their name if your template expects that convention.

4. Rename default layout to entries.html

Move layouts/default.html to entries.html (Blot’s homepage view).

5. Rename post layout to entry.html

Move layouts/post.html to entry.html (Blot’s single-post view).

6. Compile Sass and merge CSS

Convert any Sass/SCSS into plain CSS and merge into a single style.css in the template root.

7. Find and replace partials syntax

{% include navigation.html %} --> {{> navigation}}

Regex:
\{\% include (.*)\.html \%\}
{{> $1}}

Use the regex above in your editor to replace Jekyll includes with Mustache partials.

8. Fix pagination

Replace Jekyll paginator logic with Blot’s {{#pagination}} section:

<nav id="post-nav">
  {% if paginator.previous_page %}...</nav>

becomes:

{{#pagination}}
<nav id="post-nav">
  {{#previous}}
  <span class="prev">
    <a href="/page/{{previous}}" title="Previous Page">
      <span class="arrow">←</span> Newer Posts
    </a>
  </span>
  {{/previous}}
  {{#next}}
  <span class="next">
    <a href="/page/{{next}}">
      Older Posts <span class="arrow">→</span>
    </a>
  </span>
  {{/next}}
</nav>
{{/pagination}}

9. Fix menu links

Replace Jekyll’s page.url and site with Blot’s view context (e.g. {{url}}, {{#entry}}). Example:

<nav class="main-nav">
  {{^index}}<a href="/"><span class="arrow">←</span> Home</a>{{/index}}
  <a href="/about">About</a>
</nav>
