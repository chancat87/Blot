JSON feed

The JSON feed format is a syndication format, like RSS and Atom, but itâ€™s JSON instead of XML.

To expose a JSON feed on your site:

  1. Add a new view to your template.
    Warning: If you have an existing view called feed.rss or feed.xml, please name your JSON feed something other than feed.json. Two template views cannot share the same name, even if they have different extensions.
  2. Paste in the following code: 
    { 
  "version": "https://jsonfeed.org/version/1",
  "title": "{{#encodeJSON}}{{{title}}}{{/encodeJSON}}",
  "description": "{{#encodeJSON}}Feed for {{{title}}}{{/encodeJSON}}",
  "home_page_url": "{{{siteURL}}}",
  "feed_url": "{{{siteURL}}}/jsonfeed.json",
  "items": [
    {{#recentEntries}}
    { 
      "id": "{{{absoluteURL}}}",
      "title": "{{#encodeJSON}}{{{title}}}{{/encodeJSON}}",
      {{#summary}}
      "summary": "{{#encodeJSON}}{{{summary}}}{{/encodeJSON}}",
      {{/summary}}
      {{#thumbnail.large}}
      "image": "{{{siteURL}}}{{{url}}}",
      {{/thumbnail.large}}
      "content_html": "{{#encodeJSON}}{{#absoluteURLs}}{{{body}}}{{/absoluteURLs}}{{/encodeJSON}}",
      "date_published": "{{#formatDate}}YYYY-MM-DDTHH:mm:ssZ{{/formatDate}}",
      "url": "{{{absoluteURL}}}"
    }{{^last}},{{/last}}
    {{/recentEntries}}
  ]
}
