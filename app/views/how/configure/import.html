Import

Blot comes with an importer on the dashboard under Import which can import sites from the following platforms:

  • Wordpress
  • Squarespace
  • Are.na

The importer generates a folder of Markdown and image files which you can drag into your site's folder. The importer will preserve the URLs of your posts and the dates they were published.

Transferring your domain

You can switch an existing domain to Blot using the DNS records documented on the guide to using your own domain.
