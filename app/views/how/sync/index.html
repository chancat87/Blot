Change method

You can switch your site's folder from one method to another at any point. For example, if you want to use Git instead of Dropbox, Blot will seamlessly transition your folder's contents from one to the other, and you can pick up in Git where you left off in Dropbox.

Multiple sites

Each site syncs independently. You can sync one site with Dropbox and another with Git. If you have multiple sites connected to Dropbox, they can sync with the same Dropbox account or with different accounts.
