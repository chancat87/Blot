Google Docs

File extensions: .gdoc

Blot will automatically publish Google Docs inside your folder when you sync your folder with Google Drive.

Metadata

Blot reads metadata at the very top of your Google Doc. For example:

Title: October Newsletter
Tags: announcements, fall

The rest of your post goes here.

Images

Insert photos directly in Google Docs using the menu Insert then Image, or by dragging and dropping an image file into the document. If you define Alt and title text in Google Docs, Blot will use them for the image's alt text and caption.

Headings

Apply the text style Title, Heading 1, Heading 2, etc. in Google Docs to create headings.

Text formatting

Blot preserves most text formatting in Google Docs. Single line breaks are preserved by default so poetry or addresses appear as expected.

Links

Use Insert then Link to create links. Blot removes Google's tracking redirects so each link points directly to the original URL.

Lists

Use the Bulleted list, Numbered list, or Checklist buttons in Google Docs to create lists.

Tables

Use Insert then Table then select the table size to create tables.

Code

Use a single-celled tables to mark up code blocks in your document. For example:

Insert then Table then 1 x 1

If you create a double celled table, with two rows, you can specify the language of the code in the first cell to enable syntax highlighting:

Insert then Table then 2 x 1

Tip: To avoid Google Docs’ “smart quotes” (curly quotes), press Command + Z immediately after typing quotation marks. This will undo the automatic replacement of your straight quotes with curly quotes.

This follows the same convention as the Code Blocks app for Google Docs. You do not need to install the app to use code blocks in Google Docs with Blot.

Footnotes

Footnotes in your Google Doc are preserved in the published post.

Line breaks

Single line breaks are preserved by default so poetry or addresses appear as expected.

Math

You can mark up math in your Google Docs. Math is rendered by the KaTeX plugin after conversion, so behavior is consistent with other text-based formats when the plugin is enabled.

$$
f(x) = 2x^2 + 2/3
$$

Blockquotes

Use Markdown or email-style > characters for marking up block quotes. For example:

> This is a blockquote with two paragraphs.
>
> This is the second paragraph. Vestibulum enim wisi.

Comments

Comments in the document are removed from the post or page before publishing.

Columns

Google Docs cannot export columns.
