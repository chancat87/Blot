Google Docs

Supports files created by Google Docs. You must sync your folder with Google Drive in order to publish Google Docs.

Metadata

Blot reads metadata at the very top of your Google Doc. For example:

Title: October Newsletter
Tags: announcements, fall

The rest of your post goes here.

Images

Insert photos directly in Google Docs using the menu Insert then Image, or by dragging and dropping an image file into the document. They will be downloaded and stored beside the post. Alt and title text are preserved if they are part of the image in Google Docs.

Headings

Use the Google Docs heading styles (Heading 1, Heading 2, etc.) for in-post structure.

Text formatting

Bold, italic, superscript, subscript, underline and strikethrough text are preserved. Other inline styles are removed so the post inherits your theme's typography.

Links

Links in your Google Doc are cleaned automatically. Google's tracking redirects are removed so each link on your blog points directly to the original URL.

Lists

Use the Google Docs feature Insert then List to create lists. These are preserved in the published post.

Tables

Use the Google Docs feature Insert then Table to create tables. These are preserved in the published post.

Code

Use a single-celled tables to mark up code blocks in your document. For example:

Insert then Table then 1 x 1

If you create a double celled table, with two rows, you can specify the language of the code in the first cell to enable syntax highlighting:

Insert then Table then 2 x 1

Tip: To avoid Google Docs’ “smart quotes” (curly quotes), press Command + Z immediately after typing quotation marks. This will undo the automatic replacement of your straight quotes with curly quotes.

This follows the convention of and is compatible with the popular Code Blocks app for Google Docs. You do not need to install the app to use code blocks in Google Docs with Blot.

Footnotes

Footnotes in your Google Doc are preserved in the published post.

Line breaks

Single line breaks are preserved by default so poetry or addresses appear as expected.

Blockquotes

You can use Markdown or email-style > characters for block quoting. They are presented as:

> This is a blockquote with two paragraphs.
>
> This is the second paragraph. Vestibulum enim wisi.

Comments

Comments in the document are removed from the post or page before publishing.

Columns

Google Docs cannot export columns.

Size limits

Google Docs must be smaller than 5 MB. Larger documents should be split or simplified before they can be published.
